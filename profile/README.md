## What we ship

### Data Gubbins — source-agnostic data migration tooling

A product family unified by the [Frictionless Data Package](https://specs.frictionlessdata.io/data-package/) as the contract between tools:

- **[Sluice](https://github.com/caracal-lynx/sluice)** — config-driven ETL toolkit for ERP data migrations. Extract from legacy SQL / CSV / Excel / REST sources, validate against data-quality rules, transform via mappings + lookups + expressions, load to IFS, Business Central, BlueCherry, or generic CSV/JSON.
- **Spigot** *(in development)* — extract from a source → Frictionless Data Package.
- **Dredge** *(in development)* — source discovery and profiling.

Stack: TypeScript / Node.js, DuckDB staging, YAML configuration.

### BC Gubbins — Microsoft AppSource extensions for Business Central

Bolt-on extensions that make Business Central do what it ought to have done out of the box. *Enterprise software doesn't have to be complicated.*
